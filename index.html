Trust infrastructure for people and proposals

RepTag (respect tags), ScripTag (claim markup), and MeritRank (sybil-aware, provenance-weighted voting) — designed for transparent, reproducible reputation and decision-making.

Get the tools on GitHub →

Core ideas

  • Evidence-first: public feeds & consented exports → curated evidence → verifiable attestations
  • Transparent metrics & weights, reproducible reducers
  • Positive-only recognition to minimize defamation risk
  • Append-only transparency log; portable across chains

What’s live today

  • PowerShell tools to scan repos and harvest public feeds (non-scraping)
  • Claim/attestation schema stub
  • Repo scaffolding for evidence vaults per identity

Roadmap

  1. VC minting + append-only transparency log
  2. MeritRank reducer & explainer UI
  3. RepTag/ScripTag studio (inline claim tagging)
  4. API gateway + chain adapters (start off-chain)
